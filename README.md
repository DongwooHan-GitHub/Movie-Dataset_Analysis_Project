## 📖 Description
이 프로젝트는 Kaggle의 "The Movies Dataset"을 활용하여 팀원들과 함께 영화 데이터를 분석하고 유의미한 인사이트를 도출하는 것을 목표로 진행되었습니다. 각자가 설정한 분석 주제를 바탕으로 개별 분석을 수행한 후, 상호 피드백을 통해 결과를 개선했습니다. 데이터 전처리, EDA, 시각화, 모델링 등 다양한 기법을 적용해 영화 산업의 숨겨진 패턴과 트렌드를 파악하며, 이를 통해 데이터 분석 능력을 향상시키고 영화 성공 요인에 대한 깊은 이해를 얻을 수 있었습니다.
